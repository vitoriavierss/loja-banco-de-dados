# Sistema de Loja - Banco de Dados 
Criação de um esquema de Loja para a Disciplina de Banco de Dados 

## Objetivo 

Modelar e implementar um banco de dados para uma loja contendo:

- Produtos
- Fornecedores
- Vendas 

## Tecnologias

- PostgreSQL
- SQL
- DER
- Modelo Relacional

## Entidades 

- Fornecedor
- Produto
- Venda 
- Item_Venda

## Funcionalidades

= Criação do banco
- Inserção de dados
- Atualizações
- Remoções
- Consultas
